# Rakta.js fullstack backend template

This backend template mirrors the files generated by `create-rakta-app`
when **Fullstack app** is selected.

## Runtime

- Gaman.js HTTP server by default
- Optional Express, Nest, and Adonis scaffolds from the generator
- TypeScript-first source
- Feature-based modules inspired by Gaman.js enterprise structure
- Fetch-compatible request handlers behind the Gaman route layer

## Included systems

| System | Files generated |
| --- | --- |
| Environment | `src/env.ts` |
| App config | `src/config/app.config.ts` |
| Database config | `src/config/database.config.ts` |
| Database client | `src/database/client.ts` |
| Auth | `src/auth/auth.service.ts`, `src/auth/session.store.ts` |
| Security | `src/security/jwt.ts`, `src/security/password.ts` |
| Middleware | `src/middlewares/auth.middleware.ts` |
| Controllers | `src/controllers/auth.controller.ts`, `src/controllers/user.controller.ts`, `src/controllers/hello.controller.ts` |
| Services | `src/services/user.service.ts`, `src/services/otp.service.ts`, `src/services/cms.service.ts` |
| Models | `src/models/user.model.ts`, `src/models/cms-post.model.ts` |
| Routes | `src/routes/api.ts` |

## Auth endpoints

| Endpoint | Method | Purpose |
| --- | --- | --- |
| `/api/auth/login` | `POST` | Returns JWT and sets `rakta_session` |
| `/api/auth/register` | `POST` | Creates a user |
| `/api/auth/me` | `GET` | Returns authenticated user |
| `/api/auth/logout` | `POST` | Revokes current session |
| `/api/auth/forgot-password` | `POST` | Creates a password reset OTP |
| `/api/auth/reset-password` | `POST` | Resets password using OTP |
| `/api/users` | `GET/POST` | Lists or creates users |
| `/api/users/:id` | `PATCH/DELETE` | Updates or deletes users |
| `/api/cms/posts` | `GET/POST` | Lists or creates CMS posts |
| `/api/cms/posts/:id` | `PATCH/DELETE` | Updates or deletes CMS posts |
